# Olympus: AI Investment Council

Olympus is an autonomous AI-powered investment system that simulates a council of financial analysts.

## Architecture
- **Chairperson**: Orchestrates the debate and makes final decisions.
- **Analysts**:
    - **Macro Strategist**: analyzes market conditions.
    - **Technical Analyst**: looks at charts and price action.
    - **Fundamental Analyst**: reviews company financials.

Built with Python, LangGraph, and LLMs.
